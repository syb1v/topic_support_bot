# Topic Support Bot

## ⚙️ Настройка (Конфигурация)

Перед запуском бота необходимо выполнить базовую настройку.

1. **Создайте файл `.env`** в корневой папке проекта.
2. **Добавьте необходимые переменные окружения** в этот файл. Пример содержимого `.env`:

   ```env
   BOT_TOKEN=ваш_токен_бота_из_botfather
   SUPERGROUP_ID=-1001234567890
   ```

   *   `BOT_TOKEN`: Токен вашего бота, полученный от @BotFather.
   *   `SUPERGROUP_ID`: ID супергруппы, где бот будет работать (должен начинаться с `-100`).

3. **Настройка администраторов**:
   Откройте файл `config.py` и найдите строку:
   ```python
   admin_ids = [6499614618] # можно перечислить ids через запятую
   ```
   Впишите туда ID администраторов (числа), которые будут иметь доступ к админ-панели (если она есть).

## 🚀 Запуск через Docker (Рекомендуется для сервера)

Этот способ подходит для **Убунту (Linux)**, **Windows** и **macOS**, если установлен Docker.

1. **Установите Docker и Docker Compose**, если они еще не установлены.
2. Откройте терминал в папке с проектом.
3. Соберите и запустите контейнер командой:

   ```bash
   docker-compose up -d --build
   ```

   *Флаг `-d` запускает контейнер в фоновом режиме.*

4. Чтобы посмотреть логи работы бота:
   ```bash
   docker-compose logs -f
   ```
5. Чтобы остановить бота:
   ```bash
   docker-compose down
   ```

## 💻 Локальный запуск (Без Docker)

Для локального запуска вам понадобится установленный **Python 3.11** или выше.

### 🐧 Ubuntu / Linux

1. **Установите Python и venv** (если не установлены):
   ```bash
   sudo apt update
   sudo apt install python3 python3-pip python3-venv git -y
   ```

2. **Создайте виртуальное окружение**:
   ```bash
   python3 -m venv venv
   ```

3. **Активируйте окружение**:
   ```bash
   source venv/bin/activate
   ```

4. **Установите зависимости**:
   ```bash
   pip install -r requirements.txt
   ```

5. **Запустите бота**:
   ```bash
   python start.py
   ```

### 🪟 Windows

1. **Установите Python**: Скачайте и установите Python 3.11+ с [официального сайта](https://www.python.org/downloads/).
   **Важно**: При установке поставьте галочку **"Add Python to PATH"**.

2. Откройте **PowerShell** или **CMD** в папке проекта.

3. **Создайте виртуальное окружение**:
   ```powershell
   python -m venv venv
   ```

4. **Активируйте окружение**:
   ```powershell
   .\venv\Scripts\activate
   ```

5. **Установите зависимости**:
   ```powershell
   pip install -r requirements.txt
   ```

6. **Запустите бота**:
   ```powershell
   python start.py
   ```

### 🍎 macOS

1. **Установите Python** (рекомендуется через Homebrew):
   ```bash
   brew install python
   ```

2. Откройте терминал в папке проекта.

3. **Создайте виртуальное окружение**:
   ```bash
   python3 -m venv venv
   ```

4. **Активируйте окружение**:
   ```bash
   source venv/bin/activate
   ```

5. **Установите зависимости**:
   ```bash
   pip install -r requirements.txt
   ```

6. **Запустите бота**:
   ```bash
   python3 start.py
   ```

## 🛠 Полезные советы

*   **Как узнать ID группы?**
    Добавьте бота в группу, сделайте ее супергруппой (если еще нет) и перешлите сообщение из группы в бот вроде @getmyid_bot или посмотрите в логах бота (если включено логирование сообщений).
*   **Освобождение места (Docker)**:
    Если вы часто пересобираете бота, старые образы могут занимать место. Очистить неиспользуемые данные:
    ```bash
    docker system prune -a
    ```
